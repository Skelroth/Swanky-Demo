Create adF.ly, bit.ly, goo.gl short links.

 

This plugin generates a shortened link with adF.ly, bit.ly or goo.gl API for each publi link generated by an user call.

 

It's my first plugin, and it's based on "BIT.LY plugin. I hope you enjoy this plugin

 

You must have only one Shorten plugin enabled in your system!!
